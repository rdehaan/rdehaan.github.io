### *Reading project:* Algorithmic aspects of Dempster-Shafer Theory

**Description:**
[Dempster-Shafer Theory](https://en.wikipedia.org/wiki/Dempster%E2%80%93Shafer_theory) is
a formal framework for reasoning with uncertainty.
Central in this framework is
[Dempster's rule of combination](https://en.wikipedia.org/wiki/Dempster%E2%80%93Shafer_theory#Dempster's_rule_of_combination),
for combining belief functions.
One obstacle for using this framework is the computational complexity
of applying Dempster's rule of combination.
<br />
**Objectives:**
(1) Get familiar with Dempster-Shafer Theory and its relations to other frameworks such as probability theory.
(2) Make an overview of the main algorithmic and complexity results in this area,
as well as open research questions related to this.
<br />
**References:**
- J. Halpern. *Reasoning about uncertainty*. MIT press, 2003.
- P. Orponen. *Dempster's rule of combination is #P-complete*. Artificial Intelligence, 44(1&ndash;2):245&ndash;253, 1990.

### *Modeling project:* Automated reasoning and Rubik's cubes

**Description:**
[Automated reasoning](https://en.wikipedia.org/wiki/Automated_reasoning) tools
have seen enormous progress over the last few decades (e.g., algorithms for
[SAT](https://en.wikipedia.org/wiki/Boolean_satisfiability_problem) or
[ASP](https://en.wikipedia.org/wiki/Answer_set_programming)).
Such tools often work amazingly efficiently when solve computationally challenging problems.
This project involves using automated reasoning tools to answer
various questions related to Rubik's cubes, such as:
"From a given position of the cube, can we solve it in *m* steps? And how?",
"From what positions can't we solve the cube in *m* steps?"
<br />
**Objectives:**
(1) Get familiar with using automated reasoning tools
to model and solve problems.
(2) Produce a showcase demonstration of automated reasoning tools that can be used
to illustrate the use of logic in problem solving (e.g., for open days, etc).
<br />
**References:**
- M. Gebser, R. Kaminski, B. Kaufmann, and T. Schaub.
*Answer Set Solving in Practice*. Morgan & Claypool Publishers, 2012.
