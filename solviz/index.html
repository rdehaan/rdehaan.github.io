Solviz

Solviz is a tool to visualize the working of constraint-driven search algorithms. In particular, it is an application based on the clingo answer set solver that is compiled to (asynchronous) WebAssembly and that interfaces with Javascript functions that can be used to display the solver behavior (and to influence the solver speed).

The source code of Solviz is available on Github.
